# Advanced_Encryption_Standard
Advanced Encryption Standard (AES) is a specification for the encryption of electronic data established by the U.S National Institute of Standards and Technology (NIST) in 2001.

This is a simple implementation of Advanced Encryption Standard(AES-128) to encrypt and decrypt 16 byte of sample data using C++.

## AES Procedure:
**Encryption(Order)**
- Substitute Bytes
- Shift Rows
- Mix Columns
- Add Round Key

**Decryption(Order)**
- Shift Rows
- Substitute Bytes
- Add Round Key
- Mix Columns
